A) Folder Structure - Top

The content is separated from the layout.



The folder /assets, contains font, icon styles and web font files.
The folder /css, contains the responsive stylesheet + shortcode stylesheet, IE7 & IE8 stylesheets.
The folder /images contains all the images that are used in the css stylesheets or in the html pages for layout purposes.
The folder /js, contains javascript or jquery scripts and plugins ( jQuery v1.7.2 javascript framework is used + jQuery plugins + custom js scripts).
The folder /skins, contains 7 different skins' stylesheet and images.

B) HTML Structure - Top

This is a responsive layout website template.

The general template structure is the same throughout the template. Here is the general structure.



If you would like to edit the color, font or style of any elements in the document, you would need to do the following:

 a { color: #BE3939; }

To ensure that your new styles are applied, make sure that they carry enough "weight" and there isn't a style lower in the CSS file that is being applied after yours.

C) Sticky Navigation - Top


No more having to scroll up forever in order to navigate back to the homepage, as the navigation controls are now made to be sticky and that are easily accessible from anywhere on the page. The controls are now visible and easy to access even while you are moving from one headline to another.The HTML code is fairly simple to understand so we can move onto the stylesheet document.

Text color inside the about author block info that goes right after the post/gallery item can be easily altered by changing the hex value of the color rule inside the following selector:

E) Javascript - Top

All the JavaScript code responsible for extra functionality, and that are stored inside js/dw.js file. Scripts are using jQuery framework (v1.7.2+). The external plugin functions, custom javascript functions can be found in custom.js

The file starts with 'config' object, includes all the config parameters:

  1. jQuery framework (jquery.js)
  2. control-panel (control-panel.js)
  3. Layer Slider (layerslider.kreaturamedia.jquery)
  4. Isotope plugin (isotope.js)
  5. Easing (jquery.easing.1.2.js)
  6. PrettyPhoto (jquery.prettyPhoto.js)
  7. Radio button (customInput.jquery.js)
  8. For Navigation
    1. Mouse Wheel (jquery.mousewheel.js)
    2. Mobile Menu (jquery.mobilemenu.js)
    3. Visual Nav Plugin jquery.visualNav.js
